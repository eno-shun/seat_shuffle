# 席替え抽選アプリ

教育実習・学校現場向けの、GitHub Pagesで動く席替え抽選アプリです。

## ファイル
- `index.html`：生徒に見せる抽選画面
- `teacher.html`：先生用設定画面
- `common.js` / `teacher.js` / `student.js` / `styles.css`：動作に必要なファイル

## GitHub Pagesへの公開
1. GitHubで新しいリポジトリを作成します。
2. このフォルダ内のファイルをすべてアップロードします。
3. `Settings` → `Pages` を開きます。
4. `Deploy from a branch`、ブランチは `main`、フォルダは `/ (root)` を選びます。
5. 表示されたURLを開きます。

## 使い方
1. `teacher.html` を開き、座席の行数・列数、名簿・性別・最終座席を設定します。
2. 「設定を保存」を押します。
3. `index.html` を教室で表示します。
4. 抽選途中は、最終的に男子が入る席には男子、女子が入る席には女子だけがランダム表示されます。最終配置で未入力の席は空席のままです。
5. 最後の表示は先生が指定した完成配置になります。登録した生徒が全員配置されていない場合や、重複している場合は保存・抽選できません。

## 保存
設定はブラウザの `localStorage` に保存されます。同じ端末・同じブラウザでは再度開いても残ります。別端末への移行や引き継ぎには、先生画面の「JSON書き出し」「JSON読み込み」を使ってください。

## 注意
GitHub Pagesは静的サイトのため、設定をGitHub上やクラウドに自動保存しません。また、先生用画面を完全に秘匿する仕組みではありません。教室投影用の運用を想定しています。
